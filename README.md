Blog Sphere is a Java based web application built using JSP, Servlets and JDBC. It allows users to create and manage blog posts with basic user and admin functionalities.

Features:

User registration and login
User can create, edit and delete blog posts
Users can view all blogs
Users can post comments on blogs
Profile page for users
Forgot password and reset password
Admin login
Admin can view all users
Admin can delete users
Admin can view all blogs
Admin can delete blogs

Technologies Used:

Java
JSP and Servlets
JDBC
MySQL
HTML, CSS and JavaScript
Apache Tomcat
MVC architecture

Project Structure:

Java files contain all servlets and backend logic
JSP pages contain the user interface
CSS and JavaScript files handle styling and basic interactivity

How to Run:

Import the project into any Java IDE such as Eclipse
Create a MySQL database and required tables
Update the database username and password inside the project
Add the MySQL JDBC driver to the project
Run the project on an Apache Tomcat server
Open the browser and access the application
